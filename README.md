## Background

This project represents a modern "Styled Realism" web storefront built to support dynamic customer ordering and catalog browsing.

### Features
- **Dynamic Catalog**: Browse signature creations including custom cakes, sourdough breads, and hand-decorated cookies.
- **Inquiry Bag System**: A custom-built shopping cart experience designed for inquiries rather than direct transactions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Contact & Reviews**: Dedicated systems for community feedback and custom ordering inquiries.
- **Premium Aesthetics**: Modern "Styled Realism" design with smooth animations and high-fidelity visuals.

### Technical Architecture
- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Custom design system with modern properties (CSS Variables, Flexbox, Grid).
- **Vanilla JavaScript**: Lightweight, dependency-free logic for state management and UI interactions.
- **GitHub Pages**: Automated deployment via CI/CD.

For a detailed look at the design system, technical architecture, and development roadmap, please refer to [PROJECT.md](PROJECT.md).

## Install

Since this is a static web application with zero dependencies, no complex installation is required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/higherkey/amys-bakes.git
   cd amys-bakes
   ```
2. **Setup local viewing:**
   Simply open `index.html` in any modern web browser, or use a local development server like Live Server.

---

## Usage

For local development with hot-reloading:
1. Open the project in VS Code.
2. Launch the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension, or start any simple HTTP server in the directory.
3. Open the local address (e.g. `http://127.0.0.1:5500`) in your browser to test live changes.
