# Amy's Bakes ✦ Artisan Home Bakery

**[View Live Site ✦](https://higherkey.github.io/amys-bakes/)**

> **Status: Under Construction** — This project is currently a work-in-progress. All images and prices are placeholders for future availability.

Amy's Bakes is a premium, handcrafted home bakery website serving Tulsa, Oklahoma and surrounding areas. [Live Demo available here.](https://higherkey.github.io/amys-bakes/) This web application serves as a digital storefront and inquiry system for custom cakes, artisan breads, and decorated cookies.

## 🍰 Features

- **Dynamic Catalog**: Browse signature creations including custom cakes, sourdough breads, and hand-decorated cookies.
- **Inquiry Bag System**: A custom-built shopping cart experience designed for inquiries rather than direct transactions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Contact & Reviews**: Dedicated systems for community feedback and custom ordering inquiries.
- **Premium Aesthetics**: Modern "Styled Realism" design with smooth animations and high-fidelity visuals.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Custom design system with modern properties (CSS Variables, Flexbox, Grid).
- **Vanilla JavaScript**: Lightweight, dependency-free logic for state management and UI interactions.
- **GitHub Pages**: Automated deployment via CI/CD.

## 🚀 Getting Started

Since this is a static web application, no complex installation is required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/higherkey/amys-bakes.git
   ```
2. **Open the project:**
   Simply open `index.html` in any modern web browser.

For local development with hot-reloading, we recommend using [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) or a similar utility.

## 📁 Project Structure

- `assets/`: Global brand assets (logos, icons).
- `css/`: Modular stylesheets (`styles.css`, `home.css`, etc.).
- `images/`: Product photography and UI graphics.
- `js/`: Application logic (`cart.js`, `contact.js`).
- `docs/`: Technical documentation and work traces.
